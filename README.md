#__Проектная работа "Посмотри в окно"__
----------------------------------------- 
##Приложение «Посмотри в окно» — это интерактивная галерея с видеоконтентом из разных городов. В данной проектной работе выполнена стилизация всех элементов интерфейса. __Выполнена стилизация:__
* *обёртки для поискового инпута и чекбоксов*
* *текстовое поле ввода*
* *чекбоксы*
* *cостояний кнопок с помощью псевдоклассов:* Текущее (default), При наведении (:hover), В активном состоянии (:active), При фокусе (:focus и :focus-visible)
##__так же:__
* *адаптивные карточки:* Обрезка длинных заголовков с многоточием и оптимальное распределение пространства
* *позиционирование прелоадер*
##__Технологии:__
* *CSS3 — стилизация*
* *Pixel Perfect — точное соответствие макету
####Проект выполнен в рамках учебной программы по верстке.
